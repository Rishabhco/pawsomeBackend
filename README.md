
# PawSome
![logo](https://ibb.co/p0dhBPV)
Pawsome is a project that aims to provide a facilty, using which people can adopt a pet or can list a pet for adoption in an efficient way.


## 🖱 Deployment

To deploy this project run

```bash
  npm run deploy
```
